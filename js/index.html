数据类型

原始类型(primitive type): Number, String, Boolean

参考类型(reference type): Object(Function, Array, Date, RegExp)

null, undefined

数据类型的特点

  1. 弱类型 
    3 * '3'
    + '3'
    + new Date
    '000023'|0
  2. 数值型是“double-precision 64-bit format IEEE 754 values” 
    0.1 + 0.2
  3. 一切皆为对象 
    true.toString()
    (2).toString()
    '1,2,3,4'.split(',')
  4. 0, "", NaN, null, undefined为false, 其它皆为true 
    0 == ""
    !!""
  5. 类型类型的特点 
    
    var o1 = {
      a: 1
    };

    var o2 = o1;
    console.log(o2.a);

    o1.a = 2;
    console.log(o2.a);
  6. 对象都是prototype-base继承 
    
     Object.prototype.newMethod = function() {
       console.log('被污染了');
     };
     (2).newMethod();

变量

  • 全局变量
    1. 皆是window的成员
    2. 没用var声明的变量皆是全局变量
    3. 页面unload时才被GC回收
    4. 全局变量是evil
  • 局部变量
    1. 定义在函数体内
    2. 函数执行完后若未被其它对象引用侧会被GC回收

变量的作用域 - Scope Chains

  • 影响执行效率
  • 它是实现Javascript动态特性的重要机制

函数 - Function

    
    
      
  • 声明型函数
        
      
          
    1. 有特定名字
          
    2. 它位置要么在程序级的全局上下文上，要么在另一个函数体内
          
    3. 一旦进执行上下文就被创建
          
    4. 影响Variable Object
          
      
        
    
      
  • 表达式型函数
        
      
          
    1. 函数名字可选
          
    2. 出现在表达式的位置上
          
    3. 不影响Variable Object
          
    4. 在代码执行阶段创建
          
      
          会输出什么？
          
      
var foo = function _foo() {
  console.log(foo == _foo);
};

_foo();
foo();
          
      
        
    
      
  • 函名函数
         
      
           
    1. 做为对象出现在表达式位置上
           
      
           
    2. 用来避免全局变量
           
      
           
    3. 作为参数
           
      
           
    4. 闭包
           
      
         
    
      
  • 函数也是对象
      
    
      
  • 函数默认的属性和方法
        
      
          
    1. 属性：name, length, prototype
          
    2. 方法：call(scope, arg1, arg2, ...), apply(scope, [arg1, arg2,...])
      , toString(), valueOf()
        
    
    

    

  




  

    

    
对象 - Object

    
    
      
  • JS没有类的概念。一切都是对象。
      
  • JS的动态特性，对象的属性和方法可以随时添加、修改和删除
      
  • 对象可以视为包含属性和方法的哈希表
      
  • 对象默认的属性和方法：
         
      
           
    1. constructor - 指定它的构造器，用来描述如何创建一个指定的对象
           
    2. hasOwnProperty(name) - 判断是否为自己的属性
           
    3. propertyIsEnumerable(name) - 是否可用for-in枚举
           
    4. toString()
           
    5. valueOf()

this对象

      
    
      
  • 当一个函数在一个对象的上下文中被调用时，this指向这个对象
      
  • 当一个函数在外部被调用时，this指向window
      
  • 当一个函数作为构造器被调用时，this指向实例
      
    
      

    

  





  

    

      
继承

      
    
        
  • 继承就是一个对象复用其它对象的属性和方法
        
  • 继承的方式：
          
      
            
    1. 基于原型链的继承（prototype）
            
      
            
      
            
            
      
            
      原型上的成员被改动，将影响所有实例
            
      有查找时间
            
      节省内存
            
    2. 对象伪装（Object Masquerading）
            
      
            
      相当于copy，但是不会继承原型链上的属性和方法
            
      浪费内存
            
    3. 混合方式
            
      属性继承用Object Masquerading，方法的继承用prototype
